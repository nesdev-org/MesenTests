# Mesen Tests

A collection of automated tests for the [MesenCE](https://github.com/nesdev-org/MesenCE) multi-system emulator.

# Installation

This repository is designed to reside in the Tests folder within your MesenCE configuration. Clone the repository there:

```
git clone git@github.com:nesdev-org/MesenTests.git path/to/MesenCE/Tests
```

Then, launch MesenCE in Debug mode. Focus the main window, then use **Alt+F3** to run the full suite. Results are printed to the Log Window.
